About our system

The specific objectives of the Student Information Management System (SIMS) are designed to address the inefficiencies of manual processes in managing student records and to provide a modern, secure, and scalable solution tailored to the IT Department's needs. These objectives focus on leveraging contemporary technologies and best practices to create a system that ensures operational efficiency, security, and user satisfaction.

  • The SIMS will include features such as advanced search functionalities, robust data management, and document uploading, designed to support collaboration among administrators, instructors/enrollment officers, and other stakeholders.
  • The app is built using modern web technologies to ensure a fast, user-friendly, and mobile-responsive experience for all users.

Tech Stack

Tech Stack

Tools used in Development

HTML

We used HTML as the core structure for our landing page to organize text, images, and links, ensuring broad compatibility and serving as the foundation for styling and interactivity.

Java

We used Java as the primary language in developing the app because of its reliability, performance, and extensive support for Android development. Its object-oriented nature allowed for clean, maintainable code, while the large ecosystem of libraries and frameworks helped accelerate development.

NodeJS

We used Node.js as our backend because of its fast, non-blocking, event-driven architecture, which allows for efficient handling of multiple requests. Its JavaScript runtime environment enables seamless development with both the frontend and backend using the same language, streamlining the development process.

MongoDB

We used MongoDB as the database for the system because of its flexibility, scalability, and ability to handle unstructured data. Its document-oriented structure made it easy to store and query data, while its ability to scale horizontally supported the growing needs of the system.

Figma

We used Figma as our primary tool to design the app because it allows for real-time collaboration, easy iteration, and effective prototyping. Its cloud-based platform made sharing and refining designs efficient, ensuring a seamless user experience.

CSS

We used CSS to style our landing page, ensuring a visually appealing and responsive design across different devices and screen sizes.
